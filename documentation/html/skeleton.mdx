<DocsTitle href="skeleton">
# Tailwind CSS Skeleton - HTML
</DocsTitle>

The skeleton component can be used as an alternative loading indicator to the spinner by mimicking the content that will be loaded such as text, images, or video

See below our simple <Code>Skeleton</Code> example that you can use in your Tailwind CSS and HTML project.

<br />

<CodePreview component={<HTMLSkeletonExamples.DefaultSkeleton />}>
```jsx
<div class="max-w-full animate-pulse">
  <h1
    class="mb-4 block h-2.5 w-52 rounded-full bg-gray-200 font-sans text-5xl font-semibold leading-tight tracking-normal text-inherit antialiased dark:bg-gray-700">
  </h1>
  <h3
    class="mb-2.5 block h-2 w-72 max-w-[360px] rounded-full bg-gray-200 font-sans text-3xl font-semibold leading-snug tracking-normal text-inherit antialiased dark:bg-gray-700">
  </h3>
  <h3
    class="mb-2.5 block h-2 w-64 max-w-[360px] rounded-full bg-gray-200 font-sans text-3xl font-semibold leading-snug tracking-normal text-inherit antialiased dark:bg-gray-700">
  </h3>
  <h3
    class="mb-2.5 block h-2 w-72 max-w-[330px] rounded-full bg-gray-200 font-sans text-3xl font-semibold leading-snug tracking-normal text-inherit antialiased dark:bg-gray-700">
  </h3>
  <h3
    class="mb-2.5 block h-2 w-60 max-w-[300px] rounded-full bg-gray-200 font-sans text-3xl font-semibold leading-snug tracking-normal text-inherit antialiased dark:bg-gray-700">
  </h3>
  <h3
    class="block h-2 w-72 max-w-[360px] rounded-full bg-gray-200 font-sans text-3xl font-semibold leading-snug tracking-normal text-inherit antialiased dark:bg-gray-700">
  </h3>
  <p class="block font-sans text-sm antialiased font-light leading-normal sr-only text-inherit">
    Loading...
  </p>
</div>
```
</CodePreview>
